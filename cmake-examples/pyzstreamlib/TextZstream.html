class zstream.TextZstream(file, mode, bufsize)

I/O class that works with compressed (.gzip) data representing a stream of text.

Note that zopen() instead uses io.TextIOWrapper with BufferedZstream for text-mode zstreams.

Inherits ZstreamBase and io.TextIOBase

io.TextIOBase behavior:

detach()

Not supported (underlying buffers exist, but are not exposed to python)

property encoding

always "utf8"

property errors

always "ignore"
