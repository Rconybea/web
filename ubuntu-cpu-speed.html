1. Introduction

Want ability to control cpu speed to allow meaningful performance testing.

1.1. sources

org-mode source is here: ubuntu-cpu-speed-src.html

2. Prerequisites

sudo apt-get install cpufrequtils

3. Monitoring Commands

3.1. view available setters

sudo cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_available_governors

I see this output: 

performance powersave

3.2. view current scaling state

sudo cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor

I typically see this output: 

powersave

3.3. view current cpu state

cpufreq-info

4. Mode-Setting Commands

4.1. to choose performance mode

sudo cpufreq-set -c 0 -g performance

4.2. to restore powersave mode

sudo cpufreq-set -c 0 -g powersave

4.3. control core on which a task runs

cd ~/proj/perf/src
taskset -c 0 ./perf
